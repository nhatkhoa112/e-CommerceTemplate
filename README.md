## Quick start

- Install with [npm](https://www.npmjs.com/): `npm install`
- Runs the app in the development mode `npm start`
- Builds the app for production to the `build` folder `npm run build`


## What's included

- Sign In
- Forgot Password
- Home Page
- Product List (List/Grid View)
- Product Detail
- Cart
